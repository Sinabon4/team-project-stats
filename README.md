# Лабораторная работа №3 - Системы контроля версий (Git)

## 📊 Проект: Анализ курса валют

### Описание
Программа анализирует курс рубля к доллару. Она:
- Читает данные из файла `currency_data.txt`
- Выводит таблицу с курсами за каждый день
- Вычисляет скользящую среднюю за 5 дней
- Даёт прогноз курса на следующий день

---

## 🚀 Как запустить

1. Установите Python 3.x
2. Убедитесь, что файл `currency_data.txt` находится в папке с программой
3. Запустите программу командой:
   ```bash
   python main.py
   ```

## 🔀 Процесс разработки

1. Созданы ветки `dev` и личные ветки участников
2. Каждый участник внёс свой код в личную ветку
3. Созданы Pull Request'ы в `dev`
4. Возник и был успешно разрешён конфликт при слиянии
5. После завершения разработки ветка `dev` слита в `main`
